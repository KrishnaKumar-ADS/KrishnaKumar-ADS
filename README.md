## 🧠 AI Codebase Intelligence — Architecture

```
GitHub Repo
    │
    ▼ AST Parsing (1000+ files)
    │
    ▼ Gemini Embeddings → Qdrant (Vector Store)
    │                  → Neo4j  (Graph Store)
    │
    ▼ Hybrid Retrieval (semantic + dependency-aware)
    │
    ▼ Multi-LLM Router (Gemini / DeepSeek / OpenRouter)
    │
    ▼ FastAPI + Celery + Redis (async ingestion)
    │
    ▼ Next.js + D3.js  →  Real-time graph visualization ✅
```
